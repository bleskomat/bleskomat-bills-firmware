# Bleskomat V2

## Cables mapping

### TFT screen

|  ESP32  | TFT          |
|---------|--------------|
| VIN     | VCC          |
| GND     | GND          |
| CS      | GPIO5  (D5)  |
| Reset   | GPIO16 (RX2) |
| AO (DC) | GPIO17 (TX2) |
| SDA     | GPIO23 (D23) |
| SCK     | GPIO18 (D18) |
| LED     | 3.3V (3V3)   |
